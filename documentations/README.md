# Project Documentation Package

This folder organizes the project documentation requested by the project milestone brief.

## Files
- `project_documentation.md`: Main consolidated documentation aligned to milestones.
- `01_problem_and_objectives.md`: Problem statement, constraints, and objectives.
- `02_work_completed_by_milestone.md`: What is done vs. pending by milestone.
- `03_data_pipeline.md`: Data ingestion, cleaning, preprocessing, and feature engineering.
- `04_modeling_and_results.md`: LSTM and USAD modeling pipeline and reported metrics.
- `05_deployment_and_mlops_plan.md`: Deployment and MLOps implementation plan.
- `06_final_report_outline.md`: Final report and presentation structure.

## Source Notes
- Milestones source: `Docs/Project Idea.pdf`
- Problem statement source: `Docs/problem_formula.pdf` and `Docs/Problem Formulation.docx`
- Implemented work source: notebooks under `notebooks/` and generated assets under `Data/`
